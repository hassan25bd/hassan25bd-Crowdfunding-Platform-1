# Crowdfunding Platform — Server

Express + MongoDB API for the Crowdfunding Platform. See the client repo for the frontend.

## Setup

1. `npm install`
2. Copy `.env.example` to `.env` and fill in real values (MongoDB Atlas URI, JWT secret, Google OAuth client ID, Stripe test key, imgBB key).
3. `npm run dev`

Full documentation will be added as the project is built out.
